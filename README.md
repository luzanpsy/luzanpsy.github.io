# Сергей Лузан - лендинг психолога

Статичный сайт без сборки. Для публикации на GitHub Pages достаточно загрузить содержимое этой папки в репозиторий и включить Pages.

## Структура

- `index.html` - разметка и SEO-метаданные.
- `styles.css` - визуальная система, адаптив и анимации.
- `content.js` - все редактируемые тексты, цены и ссылки.
- `app.js` - отрисовка контента, FAQ, слайдер, активное меню и reveal-анимации.
- `assets/images/` - оптимизированные WebP-фотографии.

В SEO-полях сейчас указан адрес `https://luzanpsy.github.io/`. Если появится отдельный домен, замените этот адрес в `index.html`, `sitemap.xml` и `robots.txt`.
